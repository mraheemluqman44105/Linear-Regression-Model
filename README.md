Linear Regression Model - Insurance Dataset

Project Overview
This project demonstrates the implementation of a **Linear Regression Machine Learning model** to predict insurance charges based on different features such as age, BMI, smoking status, and more.



Objectives
- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess the dataset
- Apply Linear Regression model
- Analyze relationships between variables
- Predict insurance charges



Dataset Features
- Age  
- Sex  
- BMI (Body Mass Index)  
- Number of Children  
- Smoker Status  
- Region  
- Charges (Target Variable)



Technologies Used
- Python   
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  



Machine Learning Model
- Algorithm: **Linear Regression**
- Dataset split into training and testing sets
- Model trained to predict insurance charges



Key Insights
- Smokers have significantly higher insurance charges  
- Age shows a positive correlation with charges  
- BMI has a moderate impact on charges  
- Dataset is clean and suitable for modeling  


Model Evaluation
- Model performance evaluated using:
  - R² Score  
  - Mean Absolute Error (MAE)  
  - Mean Squared Error (MSE)  


Conclusion
The Linear Regression model successfully predicts insurance charges and highlights important factors such as smoking status and age. The model performs well and provides meaningful insights from the dataset.


Future Improvements
- Apply advanced models like Random Forest or Gradient Boosting  
- Perform feature engineering  
- Improve model accuracy
